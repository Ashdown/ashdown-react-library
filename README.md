Asdown React Library
====================

Sample library with sharable components for react apps

    
Run it locally
--------------

    `npm start`

Run Tests
---------

    `npm test`
    `npm run test:watch`
    `npm run test:feature`

Deploy instructions
-------------------

Build Static Files

    `./node_modules/.bin/webpack -p`

Deploy new app to heroku
    `heroku create`
    `git push heroku master`
    `heroku open`
